# 🚀 Launchpad con Rofi

```ini
nano ~/.local/share/applications/launchpad.desktop
```

```ini
nano ~/.config/rofi/launchpad.rasi
```

launchpad rasi y launhcpad desktop

## Pasos:
1️⃣ launchpad.desktop

2️⃣ rofi instalado

3️⃣ launchpad.rasi (tema o css que le dara vida o estilo)

4️⃣ Launchpad.png (opcional, es para el icono)
